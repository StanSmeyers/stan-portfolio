# Stan Smeyers portfolio - Netlify stappen

## Wat zit er in deze map?
- `index.html` = home page
- `about.html` = about me page
- `projects.html` = projects page
- `cv.html` = cv placeholder page
- `styles.css` = styling
- `script.js` = mobiele navigatie
- `assets/` = placeholder afbeeldingen

## Gratis hosten via Netlify
1. Log in op GitHub.
2. Maak een nieuwe repository, bijvoorbeeld `stan-portfolio`.
3. Upload alle bestanden uit deze map naar die repository.
4. Ga naar Netlify en log in met GitHub.
5. Kies **Add new site** > **Import an existing project**.
6. Selecteer je GitHub repository.
7. Omdat dit een statische site is, hoef je normaal geen build command in te vullen.
8. Publish de site.

### Handige instellingen
- Build command: leeg laten
- Publish directory: `.`

## Wat moet nog aangevuld worden?
- Je echte foto op `about.html`
- Je echte CV op `cv.html`
- Screenshots van je projecten op `projects.html`
- Eventueel een betere skills-sectie zodra je CV klaar is
